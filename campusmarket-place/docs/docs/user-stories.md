# User Stories — Campus Marketplace

---

## Sprint 1

## US1 — Student Registration & Login
As a student, I want to register and log in using a third-party identity provider so that my identity is verified and my account is secure.

## US2 — Role Assignment
As an admin, I want to assign roles (Student, Trade Facility Staff, Admin) so that each user has appropriate access.

## US3 — Create an Item Listing
As a student seller, I want to create a listing with photos, description, category, condition, and price so that others can buy or trade my item.

## US4 — Edit or Remove a Listing
As a student seller, I want to edit or delete my listings so that I can keep them accurate.

## US5 — Mark Listing Type
As a student seller, I want to mark a listing as For Sale, For Trade, or Either so that buyers understand the exchange type.

## US6 — Search and Filter Listings
As a student buyer, I want to search and filter listings by category, condition, and price so that I can quickly find items I want.

---

## Sprint 2

## US7 — View Seller Rating
As a student buyer, I want to view seller ratings and transaction history so that I can assess trustworthiness.

## US8 — Initiate Purchase or Trade Offer
As a student buyer, I want to initiate a purchase or trade offer so that I can start a transaction.

## US9 — Buyer-Seller Chat
As a student, I want to chat with buyers or sellers so that I can negotiate safely within the platform.

## US10 — Book Drop-Off Slot
As a student, I want to book a trade facility drop-off slot so that I can securely hand over sold items.

## US16 — Leave Rating and Review
As a student, I want to leave a rating and review after a transaction so that others can evaluate trustworthiness.

---

## Sprint 3

## US11 — Staff Confirm Item Receipt and Release
As trade facility staff, I want to confirm receipt and release of items so that transactions are properly tracked.

## US12 — Configure Facility Hours and Capacity
As an admin, I want to configure facility hours and capacity so that bookings are valid.

## US13 — Online Payment
As a student buyer, I want to pay online via a payment gateway so that the transaction is secure.

## US14 — Partial Payment with Cash Shortfall
As a student buyer, I want to make a partial payment and record cash shortfall so that I can settle later in person.

## US15 — Staff Confirm Cash Shortfall
As trade facility staff, I want to confirm cash shortfall payments so that items are only released when fully paid.

## US17 — Moderate Abusive Content
As an admin, I want to moderate abusive reviews so that the platform remains safe.

## US18 — SA Price Suggestions
As a student seller, I want to see suggested pricing based on South African data so that I can price fairly.

## US19 — Popular Categories Report
As an admin, I want to view reports of popular categories so that I understand platform usage trends.

## US20 — Trade Facility Utilisation Report
As an admin, I want to view facility utilisation so that I can manage capacity.

## US21 — Moderated Content Summary Report
As an admin, I want to view moderated content summaries so that I can monitor platform safety.

---

## Sprint 4

## US22 — Export Reports
As an admin, I want to export reports as CSV or PDF so that I can share data externally.
