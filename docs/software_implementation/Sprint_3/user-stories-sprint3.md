# User Stories — Campus Marketplace

## Sprint 3

## US11 — Staff Confirm Item Receipt and Release
As trade facility staff, I want to confirm receipt and release of items so that transactions are properly tracked.

## US12 — Configure Facility Hours and Capacity
As an admin, I want to configure facility hours and capacity so that bookings are valid.

## US13 — Online Payment
As a student buyer, I want to pay online via a payment gateway so that the transaction is secure.

## US14 — Partial Payment with Cash Shortfall
As a student buyer, I want to make a partial payment and record cash shortfall so that I can settle later in person.

## US15 — Staff Confirm Cash Shortfall
As trade facility staff, I want to confirm cash shortfall payments so that items are only released when fully paid.

## US17 — Moderate Abusive Content
As an admin, I want to moderate abusive reviews so that the platform remains safe.

## US18 — SA Price Suggestions
As a student seller, I want to see suggested pricing based on South African data so that I can price fairly.

## US19 — Popular Categories Report
As an admin, I want to view reports of popular categories so that I understand platform usage trends.

## US20 — Trade Facility Utilisation Report
As an admin, I want to view facility utilisation so that I can manage capacity.

## US21 — Moderated Content Summary Report
As an admin, I want to view moderated content summaries so that I can monitor platform safety.
